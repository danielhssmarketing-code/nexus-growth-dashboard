# Nexus Growth Dashboard V8

Esta é a versão pronta para publicar na Vercel sem precisar programar.

## Como publicar na Vercel
1. Extraia este ZIP no seu computador.
2. Entre em https://vercel.com e faça login.
3. Na tela inicial da Vercel, clique em **Add New** > **Project**.
4. Escolha a opção de enviar o projeto pelo computador (upload/import).
5. Selecione a pasta extraída deste projeto.
6. Clique em **Deploy**.
7. Quando terminar, a Vercel vai te dar um link online.

## Como instalar como aplicativo depois do deploy
### No computador
1. Abra o link da Vercel no Chrome ou Edge.
2. Procure o ícone de instalar na barra de endereço ou nos 3 pontinhos do navegador.
3. Clique em **Instalar Nexus Growth Dashboard**.

### No celular
1. Abra o link publicado no Chrome.
2. Toque nos 3 pontinhos.
3. Toque em **Adicionar à tela inicial**.
